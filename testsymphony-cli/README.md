# testsymphony-cli

## Dev

To run the CLI in dev mode, use the following command:

```shell
mvn compile quarkus:dev -Ptestsymphony-cli
```


## Build

To build the CLI, use the following command:

```shell
mvn clean package -Ptestsymphony-cli
```

## Run

To run the CLI, use the following command:

```shell
java -jar testsymphony-cli/target/testsymphony-cli-runner.jar --help
```

## Usage

```shell
java -jar target/quarkus-app/quarkus-run.jar --help
```
